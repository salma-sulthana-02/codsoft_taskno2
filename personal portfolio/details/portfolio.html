FABRICATION AND CHARACTERIZATION FOR MEASURING CO2 USING Arduino


    This project revolves around the fabrication and characterization of a CO2 measurement system using Arduino technology. The fabrication process involves assembling precise CO2 sensors, while the characterization phase focuses on calibration and optimization for accurate readings.
Leveraging Arduino's versatility, the setup ensures user-friendly integration for real-time monitoring. The project's goal is to create an accessible and cost-effective solution for measuring CO2 levels in diverse environments. By combining fabrication intricacies with meticulous characterization, the system aims to provide reliable and actionable data, contributing to environmental monitoring and sustainability efforts.



DEEP LEARNING BASED LUNG CANCER DETECTION AND CLASSIFICATION USING PYTHON


    This project focuses on revolutionizing lung cancer diagnosis through a deep learning approach. Implemented in Python, a Convolutional Neural Network (CNN) is employed to analyze medical imaging data for accurate detection and classification of lung cancer. The innovative aspect lies in the integration of a Graphical User Interface (GUI), providing a user-friendly platform to visualize results. The GUI displays classification outcomes with corresponding percentages, facilitating intuitive interpretation for medical professionals. This project not only advances the precision of lung cancer diagnosis but also prioritizes accessibility and ease of use, contributing to improved patient care and diagnostic workflows.
